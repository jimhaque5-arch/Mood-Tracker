# Public Assets

Add these files here before deploying:

- `icon.png` — 512×512 app icon (used by MiniKit + apple-touch-icon)
- `favicon.ico` — browser favicon
- `og-image.png` — 1200×630 Open Graph image (used for Farcaster frames + social sharing)

You can generate a quick og-image using Figma, Canva, or any image editor.
Use a dark background (#0A0A0A) with the ⚡ emoji and "Vibe NFT Minter" text.
